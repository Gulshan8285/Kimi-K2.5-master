Kimi K2.5

Open-source native multimodal agentic model resources, technical report, and deployment notes.

Open Report Read README Deployment Guide

Architecture

Mixture-of-Experts model with 1T total parameters and 32B activated parameters.

Context

Designed for long-context multimodal reasoning with a 256K context length.

Deployment

Includes examples for vLLM, SGLang, and KTransformers inference setups.

This Vercel deployment hosts the project documentation and report. Running the Kimi K2.5 model itself requires dedicated inference hardware and is not supported on Vercel static hosting.
